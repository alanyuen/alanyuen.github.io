Vagrant + HyperV + CentOS 開發環境設置

Vagrant CentOS 安裝

安裝 Vagrant

  • 安裝完後需要先重啟WINDOW

安裝 CentOS

  • 以管理員身份運行Console
  • mkdir demo
  • cddemo
  • vagrant init
  • vagrant box add pyranja/centos-6

  • vagrant up --provider hyperv

    問題

    如果遇到mount folder失敗情況, 可賞試修改vagrantfile, 設置 config.vm.synced_folder ".", "/vagrant", type: "smb", mount_options: ["username=YOUR_USERNAME","password=YOUR_PASSWORD"]

連接 CentOS

  • 下載PUTTY, PUTTY連接IP(vagrant up時顯示)
  • 用賬號/密碼vagrant登入
  • 用PUTTY連接, 用賬號/密碼vagrant登入

更新 CentOS

  • sudo su (轉成Root身份)
  • yum update -y

安裝 Docker

  • 用賬號/密碼vagrant登入
  • 用PUTTY連接, 用賬號/密碼vagrant登入
